# zoo_frontend

Front-end часть сервиса [зоопроект.рф](https://xn--e1adkoaagiv.xn--p1ai/).

Сайт, где пользователи могут размещать объявления о животных, ищущих дом, найденных или потерянных. Пользователи могут взаимодействовать через личные кабинеты и мессенджеры, получать уведомления и оставлять обратную связь.
