# Описание проекта "Зооплатформа" для резюме

## Проект: RESTful API для социальной платформы помощи животным "Зооплатформа"

### Достижения:

● Спроектировал и реализовал backend-архитектуру DRF-платформы с 35+ эндпоинтами, выстроив модульную структуру с 15+ взаимосвязанными моделями и полной документацией через drf-spectacular.

● Внедрил асинхронную обработку через Celery + Redis с retry-механизмом, автоматизировав email-уведомления (активация, восстановление пароля, статус куратора) и снизив время отклика UI.

● Настроил Docker-инфраструктуру с multi-stage Dockerfile и health checks, ускорив деплой в 2× и обеспечив автоматические миграции БД при запуске контейнеров.

● Оптимизировал запросы к БД через select_related/prefetch_related паттерны, сократив N+1 проблемы и ускорив загрузку вложенных данных (питомцы с фото/вакцинами) в 5 раз.

● Разработал систему управления объявлениями с назначением со-кураторов, фильтрацией по 8+ параметрам и полнотекстовым поиском, упростив координацию помощи животным.

● Имплементировал JWT-аутентификацию с blacklist-ротацией токенов и кастомную User-модель (AbstractBaseUser + PermissionsMixin), повысив безопасность API.

● Внедрил pre-commit hooks (ruff, flake8) и pytest-django, сократив code review на 60% и обеспечив стабильность релизов через автоматические тесты.

● Создал расширенную админ-панель с inline-формами, счетчиками связанных объектов и кастомными фильтрами, ускорив модерацию контента в 3 раза.

### Технологический стек:
**Backend:** Python 3.12, Django 5.0.6, DRF 3.15.2 | **Database:** PostgreSQL 15, Redis 6  
**Auth:** JWT (simplejwt 5.3.1), Djoser 2.2.3 | **DevOps:** Docker, Gunicorn 20.1.0  
**Queue:** Celery 5.4.0 | **Testing:** pytest 8.3.3, pytest-django 4.8.0  
**Code Quality:** pre-commit 3.7.1, ruff 0.5.0, flake8 7.1.1
